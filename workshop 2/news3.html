The Forger Family Takes Action in SPY x FAMILY CODE

The highly-anticipated final trailer for Spy x Family: Code White has finally dropped, featuring Spy x Family's main characters Anya, Yor and Loid Forger, alongside other important cast members including Yuri Briar, Damian Desmond, Becky Blackbell, Fiona Frost, Sylvia Sherwood and Frankie Franklin. The TOHO Animation YouTube channel released the trailer for the long-awaited movie late Sunday night, with Spy x Family: Code White set to premiere on Dec. 22, 2023 -- one of the most looked-forward-to anime releases of the year
